# ResolvIT: Autonomous IT Support Agent

ResolvIT is a modular, Python based autonomous agent architecture designed to automatically process and resolve Tier 1 and Tier 2 IT helpdesk tickets. Built to demonstrate advanced systems thinking and state management, this project ingests mock tickets formatted to mirror real world ConnectWise payloads. The agent utilizes a bespoke reasoning loop to evaluate complex technical issues and dynamically select the appropriate custom tool for resolution without relying on prepackaged low code platforms.

## Agentic Logic and Architecture

This application is defined as an agentic system because it goes beyond simple scripted automation to perform autonomous reasoning. Rather than relying on a static set of if-then statements to handle every possible scenario, the support agent uses a centralized reasoning loop to evaluate the intent and technical requirements of an incoming ticket. By analyzing the specific language within the ConnectWise notes, the agent identifies whether it needs to interact with the Active Directory tool for account management or the Knowledge Base tool for general troubleshooting. This modular design allows the agent to act as a primary decision maker that can independently select and execute the correct tool based on the unique context of the problem. This mirrors the architecture of professional agentic systems where an AI must orchestrate complex tasks and manage state across multiple external services autonomously.

The architecture is explicitly divided into specialized modules to ensure high scalability and easy integration of future tools. The agents directory houses the core reasoning logic where the support agent analyzes incoming text to determine the best course of action. The tools directory contains mock enterprise systems, including a simulated Active Directory database that accurately tracks user state and enforces realistic business rules, such as a strict three strike lockout policy for treasury accounts. 


## Local Setup and Execution
To run this application locally, you must first clone the repository and navigate into the root project directory. Create a fresh Python virtual environment to keep your system dependencies completely isolated, and then activate it using your terminal. Once your environment is active, simply execute the main Python script to watch ResolvIT autonomously ingest the JSON ticket data, evaluate the system notes, and execute the correct commands to resolve simulated Active Directory and Mitel MiCollab issues.

## Future Improvements and Scalability
The next phase for ResolvIT involves replacing the mock reasoning logic with a production grade Large Language Model integration using the Gemini CLI. By orchestrating complex agentic task execution through the Gemini ecosystem, the agent will be able to handle highly ambiguous support requests that fall outside of predefined keyword patterns. This expansion will include building bespoke reasoning loops that allow the agent to autonomously select from a broader array of tools based on real time technical documentation. Additionally, the system will be scaled to include robust Python based middleware for integrating external APIs such as the Mitel MiCollab service and live ConnectWise instances. These improvements will focus on refining non deterministic agent behaviors to ensure maximum reliability and safety while maintaining a modular architecture that supports native cloud deployment.
